# Space Game Engine

Асинхронная консольная игра на Python с использованием библиотеки curses.  
Вы управляете космическим кораблём, уворачиваетесь от мусора и с 2020 года получаете плазменную пушку.  
С каждым годом мусора становится всё больше, а сложность игры нарастает.

## Особенности

- **Асинхронная анимация** — звёзды, мусор, корабль и выстрелы работают независимо.
- **Мерцающие звёзды** — каждая звезда имеет случайную фазу и цикл яркости.
- **Физика движения** — корабль разгоняется и тормозит с инерцией.
- **Мусор на орбите** — появляется с 1961 года, частота растёт со временем (до 2020 года).
- **Плазменная пушка** — доступна с 2020 года (стрельба по пробелу).
- **Взрывы** — при попадании выстрела в мусор.
- **Отсчёт лет** — в нижней части экрана отображается текущий год и исторические события.
- **Управление** — стрелки для движения, пробел для стрельбы (после 2020), Esc для выхода.
- **Псевдографика** — символы `+*.:` для звёзд, ASCII-арт для корабля и мусора.
- **Кроссплатформенность** — работает в Linux, macOS, Windows (с `windows-curses`).

## Установка и запуск

1. Клонируйте репозиторий
2. Создайте и активируйте виртуальное окружение и установите зависимости:

```bash
uv venv
.\.venv\Scripts\Activate.ps1  # Windows  (PowerShell)
.\.venv\Scripts\activate.bat  # Windows (cmd)
source .venv/bin/activate     # Linux/macOS

uv sync                       # установить зависимости
```

3. Запустите игру:

```bash
python main.py
```

3. Управление

- **Стрелки** — перемещение корабля.
- **Пробел** — стрельба (доступна с уровня "2020 год").
- **Esc** — выход из игры.

## Как это работает

- **Игровой цикл** — `curses.wrapper(draw)` инициализирует curses, а основной цикл while coroutines продвигает все корутины через `send(None)`.
- **Корутины** — каждая анимация (звезда, мусор, корабль, выстрел) является асинхронной функцией. Управление передаётся между ними с помощью `await asyncio.sleep(0)`.
- **Тики** — один такт = `time.sleep(TIC_TIMEOUT)` (0.1 сек). Длительность фаз задаётся количеством вызовов `await sleep(ticks)`.
- **Физика** — функция `update_speed` из `physics.py` обеспечивает плавное ускорение и торможение с трением.
- **Столкновения** — проверяются через метод `has_collision` класса `Obstacle`. При попадании выстрела в мусор добавляется корутина взрыва, а препятствие удаляется.
- **Сценарий** — функция `get_garbage_delay_tics` возвращает задержку между появлением мусора в зависимости от года. Год увеличивается каждые 1.5 секунды (`YEAR_TICKS = 15`).

## Настройка

Вы можете изменить следующие параметры в `main.py`:

- `TIC_TIMEOUT` — длительность одного тика (сек).
- `YEAR_TICKS` — сколько тиков длится один год (по умолчанию 15 → 1.5 сек).
- `SYMBOLS` — набор символов для звёзд.
- `STARS_COUNT` — количество звёзд.
- `BORDER_WIDTH` — толщина рамки.

В `space_garbage.py` можно настроить скорость падения мусора (`speed`), минимальную и максимальную задержку появления.
