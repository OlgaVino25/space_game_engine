# Space Game Engine

Асинхронная консольная игра на Python с использованием библиотеки curses.
Пока что это просто звёздное небо и тестовый выстрел, но движок готов к расширению.

## Особенности

- **Асинхронная анимация** — все объекты (звёзды, выстрел, корабль) работают независимо.
- **Мерцающие звёзды** — каждая звезда имеет случайную фазу и цикл яркости.
- **Управление кораблём** — стрелки (влево, вправо, вверх, вниз), выход по Esc.
- **Выстрел — автоматический** тестовый выстрел из центра экрана.
- **Псевдографика** — символы +\*.: для звёзд, ASCII-арт для корабля.
- **Кроссплатформенность** — работает в Linux, macOS, Windows (с windows-curses).

## Установка и запуск

1. Клонируйте репозиторий и запустите:

```bash
git clone https://github.com/OlgaVino25/space_game_engine.git
cd space_game_engine
python main.py
```

2. Создайте и активируйте виртуальное окружение и установите зависимости:

```bash
uv venv
.\.venv\Scripts\Activate.ps1  # Windows  (PowerShell)
.\.venv\Scripts\activate.bat  # Windows (cmd)
source .venv/bin/activate     # Linux/macOS

uv sync                       # установить зависимости
```

2. Управление

- **Стрелки** — перемещение корабля.
- **Esc** — выход из игры.

## Как это работает

1. `curses.wrapper(draw)` — инициализирует curses и запускает основной цикл.
2. **Корутины** — каждая анимация (звезда, выстрел, корабль) — это асинхронная функция, которая управляется через `send(None)`.
3. **Тики** — один такт = `time.sleep(TIC_TIMEOUT)` (0.1 сек). Внутри корутин `await asyncio.sleep(0)` не создаёт реальной задержки, а только передаёт управление обратно в цикл. Длительность фаз задаётся количеством таких вызовов.
4. **Отрисовка** — функция `draw_frame` из `curses_tools` умеет выводить многострочный текст и стирать его (с помощью `negative=True`).

## Настройка

Вы можете изменить следующие параметры в `main.py`:

- `TIC_TIMEOUT` — длительность одного тика (сек).
- `SYMBOLS` — набор символов для звёзд.
- `SPACESHIP_SPEED` — скорость перемещения корабля (пикселей за нажатие).
